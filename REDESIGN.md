# REDESIGN.md

## Цель

Сделать современный редизайн сайта услуг аренды автовышек без поломки текущей функциональности.

## Ограничения

* Не добавлять отдельный блок цен
* Использовать существующий калькулятор как основной блок расчёта стоимости
* Не ломать PHP-логику формы, калькулятора и галереи
* Делать изменения малыми шагами

## CSS-архитектура

* `styles.css`: только глобальные переменные, reset/base, типографика, layout, кнопки, секции главной
* `form.css`: только форма и калькулятор
* `gallery.css`: только галерея
* Не использовать общие селекторы без namespace: `input`, `button`, `img`, `.card`, `.title`, `.section`
* Все крупные секции должны иметь корневой класс:

  * `.hero-section`
  * `.advantages-section`
  * `.specs-section`
  * `.use-cases-section`
  * `.gallery-section`
  * `.about-section`
  * `.service-calc`
  * `.contact-section`
  * `.site-header`
  * `.site-footer`

## Дизайн

* стиль: modern industrial / service business
* светлый фон
* тёмный текст
* акцентный оранжевый
* современная типографика
* меньше визуального шума
* аккуратные карточки и CTA

## Порядок работ

1. Исправить CSS-конфликты и слишком общие селекторы
2. Обновить `styles.css`
3. Обновить `index.php` под новые namespace
4. Обновить `form.css`
5. Обновить `gallery.css`
6. При необходимости минимально обновить `header.php`, `footer.php`, `form.php`, `gallery.php`

## Критерии приёмки

* стили не пересекаются между файлами
* калькулятор сохранён
* блок цен не добавлен
* сайт выглядит современнее
* мобильная версия не сломана
